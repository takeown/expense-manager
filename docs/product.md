# Product

## 목표
개인 사용자가 흩어진 소비 데이터를 직접 입력하거나 업로드해 이번 달 지출 흐름을 빠르게 파악하고, 절감 가능한 항목을 추천받는 AI 지출관리 앱을 만든다.

## 핵심 사용자
- 혼자 쓰는 개인 사용자
- 카드, 계좌이체, 현금 지출이 흩어져 있어 한 번에 보기 어려운 사람
- 고정비와 과소비 원인을 알고 싶은 사람

## 핵심 문제
- 소비 내역이 분산되어 월 지출 구조를 파악하기 어렵다
- 고정비 비중과 변동비 증가 원인을 직관적으로 보기 어렵다
- 절감 포인트를 스스로 찾기 번거롭다

## MVP 범위
### 입력
- 월 실수령 소득 입력
- 고정지출 입력
- 거래내역 수동 입력
- 거래내역 CSV 업로드

### 분석
- 거래 카테고리 분류
- 고정비 / 변동비 구분
- 월 총지출, 카테고리별 집계
- 지난달 대비 증감
- 절감 후보 추천

### 조회
- 이번 달 요약 대시보드
- 거래내역 리스트
- 카테고리별 지출 현황
- 자산 스냅샷
- 월별 리포트

## 제외 범위
- 금융 API 자동연동
- 실시간 카드 승인 연동
- 투자 자산 자동 수집
- 가족 공유, 멀티 유저

## 제품 원칙
- 입력은 빨라야 한다
- 모바일과 데스크탑 둘 다 우선 고려한다
- 민감정보 저장은 최소화한다
- AI 결과는 조언으로 제공한다
